# G4-1.1 Patch: GUI -> Services Facade

This patch introduces `mdr_gtk/gui_services.py` with `GUIServiceFacade` and wires
three critical GUI functions to it:

1) Import FHIR Bundle (JSON)
2) Import FHIR Package (.tgz)
3) Export Selected (JSON/XML)

This keeps Gtk callbacks thin and makes behavior testable without GTK.

## Apply
Unzip at repo root (overwrite).

## Test
python -m unittest -v
